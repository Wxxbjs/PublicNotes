# PublicNotes

存放文档

# 前言

所有的 类MarkDown文件（本质上是json，下文简称文档） 需要使用存储库附带的 **MarkDown编辑器** 进行查看

需要下载到本地进行查看

该编辑器会有 某些自定义语法 和 特别的渲染方式 ，并且维护完全个人看需求，

因此这些文档的内容可能并不适用于主流的MarkDown

# 查看方式

## 前置环境

下载 “MarkDown编辑器” 文件夹（“MarkDown轻量化嵌入.html” 可以不下）

进入该文件内部

用游览器打开 “MarkDown编辑器.html” 文件

## 文档导入

在游览器展示的编辑器界面中，点击左侧编辑区上栏的 “导入文章” 按钮

选择需要导入的文档

根据提示选择 “替换当前内容” 并 点击 “确认导入” 按钮以完成导入

# 已知问题

由于使用了 zoom 属性来实现图片的缩放，可能在某些游览器上图片的渲染并不是预期效果

但我并不打算且暂时没有办法不使用 zoom 属性

可以尝试更换别的编辑器，或者暂时忽略

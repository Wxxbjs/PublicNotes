# MarkDown编辑器

一个并不那么主流的编辑器。

# 文档的存储形式

由于需要将一些非文本的内容与文章绑定

因此文档的形式不再是 .md 文件，而是 .json

目前大致分为 主要文本内容、图片数据、预留的扩展项

# 特殊语法

除了常规的MarkDown语法，本编辑器有一些特殊的语法

## 离线图片加载

语法：`![内容](quote:图片ID)`

其中 quote 用于区分与 普通链接语法 的关键字（或者可以理解为另一种协议）

 “图片ID” 需要在 “图片管理界面” 里进行配置（点击主编辑界面上栏 “图片管理” 按钮即可进入）

该写法会将文章内置的对应ID的图片加载到文档流中进行渲染

特殊的，采用 quote协议 的图片语法的内容名会有 **参数语法** （普通链接协议没有）：

内容为：`![内容|缩放系数](quote:图片ID)`

目前只能有一个参数

缩放系数可以临时调整图片的大小（原理是直接用css属性计算）
